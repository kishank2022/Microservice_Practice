JWT === Json Web Token
xxxxx.yyyyy.zzzzz === Header.Payload.Signature
Header contains alg=HS256, type=JWT all this information
Payload contains subject(username), issuedAt, expiryDetails 
Signature contains secret key
Jwts.builder().setSubject().setIssuedAt().setExpration().signWith().compact();
here we are doing SYMMETRIC authentication because we are using one secret key for both generating and validating token.
HS256 = HMAC + SHA-256
Hash based message authentication code, secure hash algorithm with 256 bit output
